ERP - Types et Catégories

Un ERP est un Établissement Recevant du Public, il se classe en fonction de son type et de sa catégorie.

Les Types d'ERP

Le type correspond à l'activité principale de l'Établissement.

  • Type M : Magasin
  • Type N : Restaurant
  • Type O : Hôtel
  • Type P : Salle de danse, de jeux
  • Type R : Établissement d'enseignement
  • Type S : Bibliothèque, centre de documentation
  • Type U : Établissement sanitaire
  • Type W : Administration, bureau

Les Catégories d'ERP

La catégorie correspond à la capacité de l'Établissement à recevoir du public.

  • 1ère catégorie : +1500 personnes
  • 2e catégorie : de 701 à 1500 personnes
  • 3e catégorie : de 301 à 700 personnes
  • 4e catégorie : de 200 à 300 personnes
  • 5e catégorie : moins de 200 personnes
